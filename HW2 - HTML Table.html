Nate's Git Cheat Sheet

Welcome to my Git cheat sheet! Here you'll find a list of helpful git commands, their descriptions, and working examples.

List of Commands

Command Description Example
git init Creates a git repository in the current directory. git init
git add Displays the sum of all files in the repository. git add
git commit Saves changes to the local repository. (May prompt awkward conversation if repository if ready for that kind of thing.) git commit -m "I'm ready to take this to the next level."
git push Pushes saved changes down the tubes to a remote repositor. If tubes are clogged, apply pipe snake as needed. git push really hard
